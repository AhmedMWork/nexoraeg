# NEXORA V4.9 Admin Guide

## الدخول
الرابط المخفي:

```txt
/nexora-admin
/studio
/admin
```

الوضع الافتراضي Link-only لتسهيل التشغيل. للإغلاق بالكود أضف في Supabase Secrets:

```txt
REQUIRE_STUDIO_PIN=true
STUDIO_ACCESS_PIN=your-pin
```

## الصفحات
- Overview: ملخص المبيعات، الطلبات، العملاء، المخزون، التحويلات، والتنبيهات.
- Products: إضافة وتعديل المنتجات، السعر، السعر قبل الخصم، الصور، الألوان، المقاسات، المخزون، SEO.
- Orders: إدارة حالات الطلب، الدفع COD، ومتابعة العميل.
- Customers: داتا العملاء من الطلبات.
- Inventory: متابعة المخزون.
- Coupons: كوبونات الخصم.
- Limited Drops: الإصدارات المحدودة.
- Reviews: تقييمات بنصف نجمة وصور.
- Analytics: الزيارات، السلة، checkout، abandoned cart، product interest.
- SEO: ملفات Google وSearch Console.
- System Health: فحص الإعدادات والنشر.
