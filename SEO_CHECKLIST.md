# NEXORA V4.1 — SEO Checklist

## Technical SEO

- [x] `robots.txt` exists.
- [x] `sitemap.xml` exists.
- [x] Admin routes excluded.
- [x] Tracking route excluded.
- [x] Web manifest exists.
- [x] Main meta title/description improved.
- [x] Open Graph tags added.
- [x] Canonical URL added.
- [x] Error boundary prevents blank pages.

## Content SEO

- [x] FAQ page contains real questions and answers.
- [x] Shipping & Returns page contains clear policy.
- [x] Size Guide removed from navigation.
- [x] Newsletter removed from homepage.
- [x] Customer reviews are not hardcoded.
- [x] Arabic/English copy reviewed for customer-facing pages.

## Product SEO

For every product, complete:

- Product name in English and Arabic.
- Short description.
- Full description.
- Price.
- SKU.
- Availability/stock.
- At least 2-4 product images.
- Alt text for images.
- Category and gender.
