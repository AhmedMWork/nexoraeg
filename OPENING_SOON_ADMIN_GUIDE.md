# Opening Soon Admin Guide

Open:

```txt
/nexora-admin/controls
```

Choose the `Launch Mode` tab.

## Recommended flow

1. Set launch date/time.
2. Keep `Auto-open` enabled if you want the store to open automatically when the timer ends.
3. Write a premium title/subtitle.
4. Keep Notify Form enabled to collect launch subscribers.
5. Click Preview to check `/opening-soon`.
6. Enable Launch Mode.
7. Save.

## Customer behavior

When Launch Mode is enabled, customers see the Opening Soon page instead of the shop, cart, checkout, product pages, or other public routes.

## Admin behavior

Admin stays available under:

```txt
/nexora-admin
```

## Important

If you enable Launch Mode on production, test the admin route first so you can turn it off when needed.
